# Sordum.Org Utilities Instaler

Инсталлятор наиболее полезных утилит из проекта Sordum.Org

Описание утилит на английском языке смотри на сайте <https://www.sordum.org>. Там же смотри новые версии программ.

The Instaler of the most useful itilities from Sordum.Org Project is collected.

The Utilities description on English see on <https://www.sordum.org>. New Versions of programs see there also.

Build
-----

For Build the Intaller needs Wix Tolset and Wix Edit. See Chocolateey.Org for Install These Packets.

Install
-------

After Build the Installer Simple Install the Packet by sordum.org.msi
